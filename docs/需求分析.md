# 需求分析

> 对湘大搜索进行详细的需求分析，由浅入深，便于优化架构设计

## 基本需求

> 需要能对湘大所有的网站进行内容的搜索

## 细化分析

内容搜索分为几个部分：

1. 对内容进行索引
2. 对于文本子串进行中英文分词
3. 通过关键字搜索

其中：
1. 对内容进行索引分为两个部分：

    1. 爬虫爬取网站内容数据
    2. 接受内容数据后进行分类和索引
    
    管理系统负责第二部分，因此需要对数据进行详细的处理。
    
    因为在引擎中不仅存储【湘大搜索】的数据，还有其他数据，因此只能使用一个索引。
    每个文档存储的内容有全文和URL
